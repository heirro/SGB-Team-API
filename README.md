# SGB Team API

## Explanation :
WEB Based use `api.php`

CLI use `cli.php`

## Config :
- $CODE `SGB Team Member Code`
- $KEY `API KEY`
- $SECRET_KEY `Secret Key Member`

      { status: "ok" }

- Code Invalid

      { status: "fail" }
